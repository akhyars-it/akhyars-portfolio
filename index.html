ISS Tracker

International Space Station (ISS) Tracker is a web application that provides real-time tracking of the ISS's location using React for the frontend and Node.js for the backend. The application fetches live data from public APIs to display the current position of the ISS on an interactive map, along with additional information such as speed, altitude, and upcoming passes over specific locations. The user-friendly interface allows users to easily visualize the ISS's trajectory and access relevant data, making it an engaging tool for space enthusiasts and educators alike.
